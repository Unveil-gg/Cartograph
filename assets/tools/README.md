## Icons Included

| File | Icon Name | Tool |
|------|-----------|------|
| `move.png` | Move | Pan/move canvas |
| `square-dashed.png` | Square Dashed Mouse Pointer | Select rectangular region |
| `paintbrush.png` | Paintbrush | Paint tiles |
| `eraser.png` | Eraser | Erase tiles |
| `paint-bucket.png` | Paint Bucket | Fill areas |
| `map-pinned.png` | Map Pinned | Place markers |
| `pipette.png` | Pipette | Color picker/eyedropper |

## Icon Format

- **Format:** PNG with transparency
- **Size:** 64×64 pixels (recommended)
- **Color:** Monochrome/line art style
- **Background:** Transparent

## Replacing Icons

To replace any tool icon:

1. Create or download a 64×64 PNG with transparency
2. Name it exactly as shown above (e.g., `move.png`)
3. Replace the file in this directory
4. Restart Cartograph

The icon manager will automatically reload them on startup.
