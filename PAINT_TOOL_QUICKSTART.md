# Paint Tool Quick Start Guide

## Getting Started with the Paint Tool

### Switching to Paint Tool
1. Click "Paint" in the Tools panel (left sidebar), OR
2. Press **B** key on your keyboard

### Selecting a Color
When the Paint tool is active, you'll see the palette in the Tools panel:
- Click on any color swatch to select it
- The selected color will be used for painting

### Painting Tiles
**To Paint:**
- Click on any tile in the canvas
- Drag across multiple tiles to paint continuously
- The cursor shows a preview of the color you'll paint

**To Erase:**
Choose any of these methods:
1. **Right-click** on the tile
2. **Two-finger tap** on trackpad (macOS)
3. **Hold E key** + left-click

### Visual Feedback
- **Green/colored preview**: Shows the tile that will be painted
- **Red outline**: Shows the tile that will be erased
- **Status bar**: Displays current tool and shortcuts at the bottom

### Keyboard Shortcuts

#### Quick Tool Switching
- `V` - Move tool (pan and zoom)
- `B` - Paint tool
- `E` - Erase tool
- `S` - Select tool
- `I` - Eyedropper tool

#### Edit Commands
- `Cmd+Z` (Mac) / `Ctrl+Z` (Win/Linux) - Undo
- `Cmd+Y` (Mac) / `Ctrl+Y` (Win/Linux) - Redo

#### View Controls
- **Mouse wheel** - Zoom in/out
- **Middle mouse** - Pan (works with any tool)
- `=` - Zoom in
- `-` - Zoom out
- `G` - Toggle grid

## Tips

1. **Different colors per tile**: Each tile can have its own color from the palette
2. **Room boundaries**: You can only paint within defined rooms
3. **Undo/Redo**: Every paint stroke can be undone - don't worry about mistakes!
4. **Grid snapping**: The cursor automatically snaps to the grid for precision
5. **Keyboard efficiency**: Use keyboard shortcuts to switch tools quickly

## Common Workflows

### Painting a Large Area
1. Press `B` to switch to Paint tool
2. Select your color from the palette
3. Click and drag across the area
4. Use `Cmd+Z` to undo if needed

### Quick Erase While Painting
1. While in Paint tool mode
2. Hold `E` key + click to erase specific tiles
3. Or just right-click (no need to switch tools)

### Color Selection
1. Pick a color from the palette
2. Paint some tiles
3. Pick a different color
4. Paint more tiles
5. Each tile retains its color independently

## Troubleshooting

**Can't paint anything?**
- Make sure you've created a room first (use Rectangle tool)
- Check that you're clicking inside a room boundary

**Wrong color appearing?**
- Check which palette color is selected
- Click the color swatch to select a different one

**Erase not working?**
- Try right-click instead of left-click
- Or hold `E` key while left-clicking

**Keyboard shortcuts not working?**
- Make sure no text input field is focused
- Click on the canvas first to ensure it has focus
