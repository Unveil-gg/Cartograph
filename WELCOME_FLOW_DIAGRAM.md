## Key Decision Points

1. **Startup:** Welcome screen vs Editor
   - Current: Always show Welcome
   - Future: Option to skip and go to last project

2. **Project Creation:** Template vs Custom
   - Quick start: Use template
   - Advanced: Choose Custom and configure

3. **Recent Projects:** Click to open
   - Future: Right-click for options (remove, pin, etc.)

4. **Import:** File picker (stubbed)
   - Future: Drag & drop support

## Extension Points

### Adding New Templates

```cpp
// In UI.h
enum class ProjectTemplate {
    Custom, Small, Medium, Large, Metroidvania,
    YourNewTemplate  // Add here
};

// In UI.cpp::ApplyTemplate()
case ProjectTemplate::YourNewTemplate:
    newProjectConfig.cellSize = 20;
    newProjectConfig.mapWidth = 200;
    newProjectConfig.mapHeight = 150;
    break;

// In UI.cpp::RenderNewProjectModal()
if (ImGui::RadioButton("Your Template Name",
    selectedTemplate == ProjectTemplate::YourNewTemplate)) {
    selectedTemplate = ProjectTemplate::YourNewTemplate;
    ApplyTemplate(ProjectTemplate::YourNewTemplate);
}
```

### Adding Recent Projects Storage

```cpp
// In UI.cpp::LoadRecentProjects()
std::string configPath = Platform::GetUserDataDir() + "recent.json";
nlohmann::json j = IOJson::LoadJsonFile(configPath);

for (auto& item : j["projects"]) {
    RecentProject rp;
    rp.path = item["path"];
    rp.name = item["name"];
    rp.lastModified = item["lastModified"];
    if (Platform::Fs::FileExists(rp.path)) {
        recentProjects.push_back(rp);
    }
}
```
